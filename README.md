# 🛒 Onin Shop Stock Management System

ระบบจัดการสต๊อกสินค้าและบริหารข้อมูลร้านค้า พัฒนาขึ้นเพื่อใช้เป็นผลงานด้าน Web Development และ Database Management โดยมุ่งเน้นการจัดการข้อมูลสินค้าให้มีประสิทธิภาพ ใช้งานง่าย และรองรับการขยายระบบในอนาคต

## 📌 Project Overview

Onin Shop Stock Management System เป็นเว็บแอปพลิเคชันสำหรับจัดการสินค้าในร้านค้า ช่วยลดปัญหาการบันทึกข้อมูลแบบเดิม เพิ่มความสะดวกในการตรวจสอบจำนวนสินค้า และช่วยให้ผู้ดูแลระบบสามารถบริหารจัดการข้อมูลได้อย่างเป็นระบบ

โปรเจกต์นี้ถูกพัฒนาขึ้นเพื่อฝึกทักษะการพัฒนาเว็บไซต์แบบ Full Stack ตั้งแต่การออกแบบฐานข้อมูล การเชื่อมต่อระบบหลังบ้าน ไปจนถึงการออกแบบหน้าจอผู้ใช้งาน

## 🚀 Key Features

- 🔐 ระบบเข้าสู่ระบบ (Login / Logout)
- 👤 จัดการข้อมูลผู้ใช้งาน
- 📦 เพิ่ม / แก้ไข / ลบข้อมูลสินค้า
- 🗂️ จัดการหมวดหมู่สินค้า
- 📊 รายงานจำนวนสินค้าในคลัง
- 🔍 ค้นหาข้อมูลสินค้า
- 📈 Dashboard แสดงข้อมูลภาพรวม
- 🎨 รองรับการใช้งานผ่าน Web Browser

## 💡 Purpose of This Project

โปรเจกต์นี้จัดทำขึ้นเพื่อพัฒนาทักษะด้านการเขียนโปรแกรมและสร้างระบบที่สามารถนำไปประยุกต์ใช้กับธุรกิจร้านค้าจริงได้ เช่น ร้านขายสินค้า ร้านมินิมาร์ท หรือระบบคลังสินค้าเบื้องต้น
